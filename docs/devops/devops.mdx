### O que é DevOps
Os softwares estão em constantes mudanças, a área de desenvolvimento depende das decisões de negócio para iniciar algum desenvolvimento,
quando há alguma decisão de construir alguma aplicação, ao finalizar ou realizar algum teste, é preciso torna-las disponíveis, para isso
é necessário do time de operações. O DevOps é a união de práticas de desenvolvimento e operações usando diversas tecnologias e uma delas
é IaC (Infraestrutura como Código).

### IaC (Infraestrutura como Código)
A ação de tranformar a infraestrutura como código, atráves de scripts, seja em um ambiente na nuvem ou on premise, podemos criar essas infraestruturas com
ferramentas como bash ou powershell, como também usar o terraform ou ansible. As nuvems possuem suas próprias ferramentas no caso da
Azure o ARM Template. Podemos gerenciar e escalar varios serviços apenas com simples script em alguns segundos.

### Benefícios da IaC

- Automatizar a infraestrutura.
- Tempo de entrega de ambiente mais rápido.
- Menos documentação e manuais sobre a infraestrutura.
- Código versionado
- Código é do negócio, deixando o código da infraestrutura compartilhavem entre os colaboradores.
